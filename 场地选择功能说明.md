# 场地选择功能说明

## 功能概述

鹊桥AI婚礼帮手的场地选择功能提供了全面的婚礼场地搜索、对比、预约和管理解决方案，帮助新人找到理想的婚礼举办地。

## 核心功能

### 1. 智能搜索系统
- **关键词搜索**: 支持场地名称、地点、特色等多维度搜索
- **分类筛选**: 8大场地分类，精准定位需求
- **高级筛选**: 价格范围、评分、容量等多条件筛选
- **智能排序**: 评分优先、价格优先、名称排序

### 2. 场地分类体系
- **🏨 酒店婚礼**: 酒店宴会厅和婚礼套房
- **🌳 花园户外**: 花园、草坪、户外场地
- **🏖️ 海滨沙滩**: 海滨、沙滩、水上婚礼
- **⛪ 教堂仪式**: 教堂、宗教仪式场所
- **🍽️ 餐厅会所**: 餐厅、会所、私人场所
- **🏰 古堡庄园**: 古堡、庄园、历史建筑
- **🏢 现代场馆**: 现代展馆、艺术空间
- **✨ 特色场地**: 游乐园、博物馆等特色场所

### 3. 场地信息展示
- **图片展示**: 高质量场地图片和全景
- **详细信息**: 容纳人数、价格范围、评分等
- **特色标签**: 快速了解场地特色和服务
- **联系信息**: 电话、邮箱、官方网站等

### 4. 多维度对比功能
- **表格对比**: 直观对比多个场地信息
- **项目对比**: 评分、容量、价格、设施等对比
- **灵活选择**: 支持添加/移除对比项目
- **一键决策**: 基于对比结果做出选择

### 5. 场地详情页面
- **分页展示**: 概览、设施、套餐、评价四大板块
- **深度信息**: 详细的场地介绍和设施说明
- **套餐选择**: 基础、标准、豪华三种套餐
- **互动体验**: 图片轮播、视频预览等

### 6. 预约系统
- **三步预约**: 婚礼信息 → 套餐选择 → 特殊需求
- **进度指示**: 清晰的预约进度和步骤
- **信息验证**: 完整的预约信息确认
- **即时反馈**: 预约提交后的即时反馈

### 7. 收藏管理
- **一键收藏**: 快速收藏喜欢的场地
- **收藏列表**: 独立的收藏场地浏览
- **持久存储**: 收藏信息本地保存
- **快速筛选**: 只看收藏场地的筛选选项

## 使用流程

### 1. 搜索场地
1. 输入关键词进行搜索
2. 选择场地分类进行筛选
3. 使用高级筛选精确匹配
4. 选择排序方式查看结果

### 2. 浏览对比
1. 浏览场地列表或网格视图
2. 收藏感兴趣的场地
3. 选择多个场地进行对比
4. 查看对比表格做出决策

### 3. 预约看场
1. 查看场地详细信息
2. 选择合适的婚礼套餐
3. 填写预约信息
4. 提交预约等待联系

## 数据字段

### 场地信息
- **基本信息**: 名称、描述、地址
- **联系方式**: 电话、邮箱、网站
- **价格信息**: 价格范围、套餐价格
- **容量信息**: 最大容纳人数、座位安排
- **评分信息**: 用户评分、评价数量
- **设施服务**: 停车、WiFi、化妆间等

### 预约信息
- **婚礼信息**: 日期、人数、时间
- **联系信息**: 姓名、电话、邮箱
- **套餐选择**: 基础、标准、豪华
- **特殊需求**: 饮食要求、装饰风格等

## 使用建议

### 搜索技巧
1. **关键词**: 使用具体的地标或区域名
2. **分类筛选**: 根据婚礼风格选择分类
3. **价格筛选**: 合理设定预算范围
4. **评分筛选**: 优先查看高评分场地

### 对比策略
1. **选择同类**: 对比同类型场地
2. **关注重点**: 对比关键决策因素
3. **实地考察**: 对比后实地看场
4. **综合评估**: 不只看价格，考虑整体价值

### 预约准备
1. **信息准确**: 确保联系信息正确
2. **日期灵活**: 准备几个备选日期
3. **问题清单**: 准备要问的问题清单
4. **需求明确**: 明确特殊需求和偏好
