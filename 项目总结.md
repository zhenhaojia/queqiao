## 🚀 启动指南

### 1. 安装依赖
```bash
npm install
# 或者运行 install.bat
```

### 2. 配置环境变量
复制 `.env.local` 文件并配置数据库等信息

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问应用
打开浏览器访问 [http://localhost:3000](http://localhost:3000)
